# 用户
分为游客，普通用户和admin (管理员)

# 账号管理
游客无需注册。注册的只能是普通用户。admin系统内置。
用户名的长度需为4至30个字符，只允许使用汉字(1个汉字=2个字符)、数字 (0-9)、字母 (a-z, A-Z)、下划线 (_)、点 (.)、短横线 (-)。密码必须是7-16位英文字母、数字、字符组合（不能是纯数字)。
普通用户可以删除自己账户，并清理所有私有数据。

# 字集
管理员管理所有允许的字(包括标点)，而且公开。
普通用户通过采集功能，有自己的书写(处于未精修状态)。接着可以进行调整，然后设置成已精修状态。
一个字可以有多个书写。

# 作品集
普通用户可以添加作品(不包含书写)，让admin审核。审核完成前状态是审核中。审核通过后，该作品状态是公开，所有用户可见。作品一旦公开，admin之外无权修改或删除。
所有的作品，自动处于这个普通用户的未精修状态。接着可以调整，主要是选择多个书写中的一个，然后设置成已精修状态。
admin可以管理所有作品，注意只包含内容。admin不拥有任何书写。

# 公开
普通用户在设置里，可以把书写设成公开 (注意只有一个选项)。一旦公开，设成已精修的字和作品，都将公开。注意只有已精修的才公开。
一旦把自己的书写设成不公开，那么个人全部的字和作品都将对其他人不可见，也就不能评分。以前的评分仍然保留。
作品有3个状态：审核中，审核不通过，公开。作品不包含书写。
书写公开，是指普通用户公开自己的书写。已精修且处于公开状态的书写，所有普通账户和游客可见，并且可以评分。

# 评分
公开的书写，进入评分系统。其他普通用户在浏览某个字或者作品时，可以进行打分。打分为10分制。
游客可以浏览公开的字和作品，但不能打分。
浏览公开字或者作品时，按照平均分从高到低排列。

# 界面
## admin
admin负责管理，他不拥有任何书写。
### 字集
只有所有字。
### 作品集
有公开作品集和待审核作品。
### 设置
不用"字集和作品集可见"选项

## 普通用户
### 字集
包含全部，已精修，未精修，未采集。
### 作品集
包含全部，已精修，未精修，未采集。
### 采集
采集书写
### 设置
包含"字集和作品集可见"选项

## 游客
### 字集
只有全部
### 作品集
只有全部
